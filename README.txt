# Mini_Projet_Reseau
# Mathieu Boireau, Thibault Fouchet, Sébastien Prunier
 
Executer le code :
Faire d'abord un javac *.java
Puis lancer le serveur sur un terminal : java ServeurCloud
Lancer le client sur un autre terminal/pc : java ClientCloud (localhost/ip pc) 8000
Exemple : java ClientCloud localhost 8000
Vous pouvez créer un compte ou vous connecter a un compte déjà existant
Le compte par défaut est : login : admin
						   mdp : admin

Ce qui fonctionne par rapport au projet de départ :
Toutes les fonctionnalités établies lors de la préparation du projet sont fonctionnelles.

Possibles améliorations :
Éditer un fichier en ligne
Supprimer un compte
Consulter un fichier en ligne
Renommer un fichier 
Empêcher qu'un utilisateur se connecte depuis 2 pc en même temps.
Créer des sous-répertoires
Déplacer des fichiers dans des sous-répertoires
